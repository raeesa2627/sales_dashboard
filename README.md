# Sales Dashboard

This Sales Dashboard is a web-based application for visualizing and analyzing sales data.

## Features
- Interactive sales dashboard
- Multiple chart visualizations
- Responsive layout
- Basic filtering for insights

## Deployed Link
https://sales-dashboard-gamma-jet.vercel.app/

## Tech Stack
- React
- TypeScript
- Tailwind CSS
- Recharts

## Setup
1. Clone the repository
2. Run `npm install`
3. Start with `npm run dev`
